# 🎓 Sistema de Control de Alumnos — CBTIS 256

> Proyecto escolar de Base de Datos No Relacionales (Fase 1) — CBTis 256
> **Estado: ✅ Completado**

Este proyecto es un sistema de control de alumnos desarrollado con **PHP y MySQL**, creado como trabajo de la materia de Base de Datos No Relacionales en el CBTis 256. El sistema permite gestionar información básica de alumnos desde una interfaz web local.

---

## ⚙️ Requisitos

- [XAMPP](https://www.apachefriends.org/) instalado en tu equipo
- Navegador web (Chrome, Firefox, Edge, etc.)
- El archivo `.sql` incluido en el repositorio para importar la base de datos

---

## 🚀 Instalación y uso local

1. **Clona o descarga** este repositorio en tu equipo.
2. **Copia la carpeta del proyecto** dentro del directorio `htdocs` de XAMPP.
   Ruta típica: `C:\xampp\htdocs\`
3. **Abre el Panel de Control de XAMPP** e inicia los módulos **Apache** y **MySQL**.
4. **Importa la base de datos:**
   - Entra a `http://localhost/phpmyadmin`
   - Crea una base de datos nueva
   - Selecciónala y haz clic en **Importar**
   - Selecciona el archivo `.sql` incluido en el repositorio y confirma
5. **Accede al sistema** en tu navegador:

http://localhost/[nombre-de-la-carpeta]
